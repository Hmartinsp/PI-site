O que é Docker?

Docker é uma plataforma de código aberto que permite que você crie, implante e execute aplicativos em containers. Um container é uma unidade leve e portátil que agrupa tudo o que uma aplicação precisa para funcionar, incluindo o código, bibliotecas e dependências.

Como o Docker funciona:

O Docker cria containers a partir de imagens, que são como moldes para o ambiente de execução da aplicação.

Uma vez que o container é criado, ele pode ser executado em qualquer máquina que tenha o Docker instalado, independentemente do sistema operacional ou das configurações do computador. Isso torna o Docker extremamente útil para desenvolvedores e equipes que trabalham em ambientes diferentes.
