Tecnologias e Linguagens Estudadas

Neste Site iremos sitar algumas tecnologias usadas na formtação e criação de sites.

Nome Descrição
HTML Linguagem de marcação usada para estruturar páginas web.
CSS Usado para estilizar páginas web, definindo cores, layouts e fontes.
JavaScript Linguagem de programação que adiciona interatividade às páginas.
FTP Protocolo usado para transferir arquivos entre computadores na internet.
Docker Plataforma para criar, distribuir e rodar aplicativos em containers.
